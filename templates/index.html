Technologies Used :

Artificial Intelligence & Machine Learning

PyTorch deep learning framework with ResNet-18 CNN architecture for image classification. Trained on 5 leaf disease classes with transfer learning techniques.

Computer Vision

OpenCV and PIL libraries for image processing, preprocessing, and feature extraction. Image transformations including resizing, normalization, and augmentation.

Cloud Computing & Storage

Firebase Firestore NoSQL database for scalable data storage and real-time synchronization. Cloud-based deployment with automatic scaling capabilities.

Cybersecurity & Data Protection

SHA-256 cryptographic hashing for image integrity verification, secure session management with Flask-Session, and API authentication with secure tokens.

Blockchain Technology

Custom blockchain implementation with proof-of-work consensus for immutable disease diagnosis records and transparent audit trails.

Web Framework & Backend

Flask micro-framework with RESTful API endpoints, Jinja2 templating engine, and Werkzeug WSGI server for robust web application development.

Frontend Technologies

Bootstrap 5 responsive CSS framework, Font Awesome icons, Google Fonts typography, and vanilla JavaScript for interactive user interfaces.

Enterprise Resource Planning (ERP)

Integrated ERP system with automated record management, analytics dashboard, and comprehensive reporting for agricultural data management.

Data Analytics & Visualization

Real-time disease distribution charts, predictive analytics for disease progression, and statistical analysis of diagnosis patterns.

Responsive Design

Mobile-first responsive design with adaptive layouts, touch-friendly interfaces, and cross-device compatibility for farmers and agricultural professionals.
